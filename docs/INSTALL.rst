eval.theme Installation
-----------------------

To install eval.theme using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``eval.theme`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        eval.theme

* Re-run buildout, e.g. with:

    $ ./bin/buildout
